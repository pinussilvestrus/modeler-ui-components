# modeler-ui-components

Reusable components and styles, for Camunda Modeler.

## Installation

```sh
git clone https://github.com/pinussilvestrus/modeler-ui-components.git
cd modeler-ui-components
npm install
```

## Build that project

```sh
npm run build
```

## Start the example

```sh
cd example
npm run dev
```
